# Carcassonne-Visualiser

This application requires: 
- raylib-go (https://github.com/gen2brain/raylib-go/tree/master),
- SDL2

## Run application

### Linux

```console
./make.sh run _log-filename_
```

### Windows

```console
./make.ps1 run _log-filename_
```

## Update requirements

### Linux

```console
./make.sh update
```

### Windows

```console
./make.ps1 update
```
